# 项目简介
### **一、图片转字符画**
##### *环境要求*
```
Package         Version
--------------- --------
Pillow          5.3.0
```
##### *实验原理*
* 获取图片的每一个像素，获取像素值（r,g,b,alpha）
* 将其利用灰度公式转换成灰度值，在字符列表中找到对应的字符（灰度值大的亮，灰度值小的比较暗）
* 完毕
